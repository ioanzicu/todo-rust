# Oh My Gorgeous Todo APP in Rust

An amazing to-do web app in Rust using Actix Framework!


- `/backend/`

    ```bash
    cd backend
    ```


    ```bash
    cargo run
    ```

- `/frontend/`

    ```bash
    cd frontend
    ```


    ```bash
    npx run dev
    ```


## Version 0.01
